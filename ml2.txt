⚙️ Steps of ID3 Algorithm
Start with full dataset (S)
If all examples belong to same class
Return a leaf node
If no attributes left
Return majority class
Select best attribute
Calculate Information Gain for all attributes
Choose attribute with highest gain
Create decision node
Split dataset
For each value of attribute → create branch
Repeat recursively
Apply ID3 on each subset

💻 Simple Python Code (Basic ID3)
import math
from collections import Counter

def entropy(data):
    labels = [row[-1] for row in data]
    total = len(labels)
    counts = Counter(labels)
    
    ent = 0
    for c in counts.values():
        p = c / total
        ent -= p * math.log2(p)
    return ent


def info_gain(data, attr_index):
    total_entropy = entropy(data)
    values = {}
    
    for row in data:
        key = row[attr_index]
        values.setdefault(key, []).append(row)
    
    weighted_entropy = 0
    for subset in values.values():
        weighted_entropy += (len(subset)/len(data)) * entropy(subset)
    
    return total_entropy - weighted_entropy


def best_attribute(data):
    gains = [info_gain(data, i) for i in range(len(data[0]) - 1)]
    return gains.index(max(gains))
